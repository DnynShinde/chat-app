# chat-app

### chat-application, start chat in room or in group
    Share link with your friends, let's start the conversation.

1.NodeJS
1.Express JS + Socket.io

>Hosted at: [Chat App](https://dbs-chat-app.herokuapp.com/) 


Install dependencies:

```bash
npm install 
```

Run server:

```bash
npm run start
npm run dev
```
